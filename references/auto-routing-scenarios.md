# Auto Routing Scenario Library

Runtime routing data consulted by `commands/auto.md`. Each case is a routing contract for `/aaron-marketing:auto` or `/aaron-marketing:auto --deep`, using the same `eval-case` schema as `evals/<skill-name>/cases.md`, plus `scenario_family`, `risk_gates`, `expected_route`, `blocking_inputs`, and `must_not` fields. Scenarios cover all three disciplines: SEO/GEO routes through `/aaron-marketing:seo-geo` modes, influencer through `/aaron-marketing:impact` phases, paid ads through `/aaron-marketing:paid` phases (boundary rules in [aaron-product-api-contract.md](aaron-product-api-contract.md)).
